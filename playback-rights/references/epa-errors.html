Esta función está disponible para un conjunto específico de clientes con acceso a la fase de disponibilidad limitada del servicio de gestión de derechos de reproducción.

Referencia de error

La siguiente lista de estado y códigos de error lo ayudarán a depurar problemas cuando trabaje con el servicio de administración de derechos de reproducción de Brightcove:

Estado Código de error Subcódigo de error Descripción
400 BAD_REQUEST DUPLICATE_PARAMETERS Se proporcionó el mismo nombre de parámetro más de una vez en la solicitud.
401 ACCESS_DENIED TOKEN_REQUIRED A Token web JSON (JWT) es un requisito de los derechos de reproducción, pero no se proporcionó.
401 ACCESS_DENIED TOKEN_VALIDATION los Token web JSON (JWT) proporcionado no pasó la validación.
403 FORBIDDEN ACCOUNT_ID El ID de cuenta en la clave de política no coincide con la cuenta en la solicitud de API.
403 FORBIDDEN CLIENT_GEO El video no puede reproducirse en la región geográfica actual. El mensaje contendrá información adicional sobre el problema específico.
403 FORBIDDEN CLIENT_IP El video está restringido a la dirección IP actual.
403 FORBIDDEN DOMAIN El video no se puede reproducir en el dominio actual.
403 FORBIDDEN OFFERS los Token web JSON (JWT) no permite la reproducción según las ofertas disponibles en el video.
403 FORBIDDEN TAGS los Token web JSON (JWT) no permite la reproducción basada en etiquetas disponibles en el video.
403 FORBIDDEN VIDEO_NOT_PLAYABLE Para una sola solicitud de video, el video existe, pero no se puede reproducir ahora.
404 NOT_FOUND VIDEO_NOT_FOUND El video solicitado no está disponible.
404 NOT_FOUND PLAYLIST_NOT_FOUND La lista de reproducción solicitada no está disponible.
405 METHOD_NOT_ALLOWED Solo se permiten GET, HEAD y OPTIONS para esta API.
500 SERVER_ERROR Error de servidor interno.
502 SERVER_ERROR ACCOUNT_RETRIEVE_FAILURE Recibí una mala respuesta de un servidor backend.
502 SERVER_ERROR VIDEO_RETRIEVE_FAILURE
502 SERVER_ERROR VIDEO_URLS_RETRIEVE_FAILURE
502 SERVER_ERROR PLAYLIST_RETRIEVE_FAILURE
502 SERVER_ERROR PLAYBACK_RIGHT_RETRIEVE_FAILURE No se puede recuperar la reproducción directamente desde CDN.
502 SERVER_ERROR PLAYLIST_VIDEOS_RETRIEVE_FAILURE
502 SERVER_ERROR LICENSE_RETRIEVE_FAILURE No se pudo obtener la licencia del servidor de licencias para este video.
502 SERVER_ERROR OFFERS_RETRIEVE_FAILURE No se pudieron obtener ofertas de la API para este video.
502 SERVER_ERROR RIGHTS_RETRIEVE_FAILURE No se pudieron recuperar los derechos de reproducción de este video.
503 SERVICE_UNAVAILABLE Devolvió esta respuesta de un servidor backend.
504 SERVER_TIMEOUT Se agotó el tiempo de espera de un servidor backend o de uno de los servidores en los que dependen.
