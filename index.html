Documentación de API de plataforma de Video Cloud

Con Video Cloud, puede ofrecer experiencias rápidas, innovadoras y similares a la TV, insertar dinámicamente anuncios del lado del servidor para maximizar los ingresos y llegar a sus audiencias dondequiera que se encuentren.

Las API de la plataforma son API RESTful que le permiten acceder a casi todos los aspectos de la plataforma de Video Cloud mediante programación. Normalmente, las API se utilizan para crear integración entre su CMS u otro sistema, o para crear aplicaciones personalizadas que respondan a las necesidades específicas de sus usuarios.

Tenga en cuenta que este sitio contiene documentación para las API principales de Video Cloud.
