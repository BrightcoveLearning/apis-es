Descripción general

Este documento detalla las operaciones disponibles desde diferentes API. Los necesita cuando realiza solicitudes al servicio OAuth para credenciales del cliente.

Nota: para algunas API, las credenciales de cliente también se pueden obtener mediante la IU de credenciales de OAuth. Consulte el siguiente documento para obtener instrucciones sobre el uso de la interfaz de usuario de OAuth.

  • Administrar las credenciales de autenticación de API
