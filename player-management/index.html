La gestión de reproductores se refiere a la creación, edición y gestión de reproductores como un recurso para los editores. Player Management se logra mediante una API REST del lado del servidor que permite a sus consumidores crear, configurar, previsualizar y publicar instancias de reproductor.

Tenga en cuenta que la mayoría de las operaciones que se pueden realizar con la API también se pueden realizar en Estudio.
