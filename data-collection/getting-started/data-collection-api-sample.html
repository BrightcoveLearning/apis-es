Notas sobre la muestra

El código para esta muestra es todo JavaScript, y hay dos partes (ambas se encuentran en CodePen: el script del complemento está en la parte superior y un fragmento corto de código que agrega el complemento al reproductor en la parte inferior):

  • El script que agrega el complemento al reproductor videojs.
  • El script para el complemento que envía datos al recopilador de datos

Tenga en cuenta que la forma en que se "envían" los datos a la API de recopilación de datos es estableciendo la URL de la solicitud como src valor para una nueva imagen (img ) elemento y luego inyectar ese elemento en la página.

Debería utilizar el mismo método en lugar de XMLHttpRequest o alguna otra para evitar CORS asuntos.
