Archivos WebVTT no válidos

Si ingiere un archivo WebVTT no válido mediante la API de ingesta dinámica, la operación se realizará correctamente y el archivo se agregará como un activo al video. Sin embargo, los subtítulos o capítulos no funcionarán en Brightcove Player y evitarán que otros archivos WebVTT válidos asociados con el video también funcionen.

Solución alterna

Puede validar sus archivos WebVTT usando este validador en línea.
