Error de mensajes

Los siguientes mensajes de error pueden ser devueltos por la API de ingesta dinámica

Mensajes de error de ingesta dinámica
Código Mensaje Tipo Descripción
400 PROFILE json

No se puede encontrar el perfil por nombre
400 BAD_REQUEST json

No se puede analizar el cuerpo de la solicitud.
400 NOT_SUBMITTED json

No se puede enviar el trabajo. Vuelve a intentarlo más tarde.
400 NO_SUCH_VIDEO json

No se puede encontrar el video al que se hace referencia
400 NO_SOURCE json

No se puede encontrar una fuente para usar
400 CDN_CREDENTIALS json

No se pueden recuperar las credenciales de CDN
400 BAD_CALLBACKS json

Las devoluciones de llamada no tenían el formato esperado
401 UNAUTHORIZED json

Error de autenticación; Verifique que las credenciales de su cliente sean correctas para el token de acceso.
403 DYNAMIC_DELIVERY_NOT_ALLOWED json

Esta cuenta no está habilitada para Dynamic Delivery, pero se especificó un perfil de Dynamic Delivery
403 CONTEXT_AWARE_ENCODING_NOT_ALLOWED json

Esta cuenta no está habilitada para codificación sensible al contexto, pero se especificó un perfil de codificación consciente del contexto.
404 RESOURCE_NOT_FOUND json

La api no pudo encontrar el recurso que solicitaste
415 UNSUPPORTED_MEDIA_TYPE json

O el archivo que está intentando ingerir no es de un tipo compatible (consulte Formatos, códecs y contenedores de audio y video compatibles ) o la solicitud no incluía un Content-Type: application/json encabezamiento
422 MALFORMED_SOURCE_URL json

La URL de origen tiene un formato incorrecto.
422 BAD_PROTOCOL_SOURCE_URL json

La URL de origen utiliza un protocolo no compatible.
422 EXCEED_MAXIMUM_VTT_SOURCES json

Las fuentes vtt superan el tamaño máximo.
422 INVALID_VTT_KIND json

El tipo vtt no es válido.
422 CONSTRAINT_VIOLATION json

No se permite la captura de imagen si se proporciona una fuente de imagen.
422 UNPROCESSABLE_ENTITY json

Los datos de la solicitud contienen alguna entidad no procesable.
429 RATE_LIMIT_EXCEEDED json

No se ha creado el trabajo de Ingesta dinámica. Reduzca la cantidad de trabajos simultáneos para esta cuenta antes de volver a intentarlo. Este error ha sido registrado
400 CDN_CONFIGS json

No se pueden recuperar las credenciales de CDN
400 AMBIGUOUS_REQUEST json

En la solicitud se establecieron tanto una URL maestra como use_archived_master.
500 INGEST_ERROR json

Este error generalmente indica que hubo una interrupción temporal del servicio de ingesta.
