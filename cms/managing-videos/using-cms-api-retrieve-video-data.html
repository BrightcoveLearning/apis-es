Introducción

En este tema, aprenderá a recuperar varios tipos de información sobre sus videos de Video Cloud, utilizando API de CMS peticiones. Para crear estas solicitudes, agregue los puntos finales enumerados en este documento a la URL base con su ID de cuenta.

    https://cms.api.brightcove.com/v1/accounts/{account_id}/{endpoint}

Solicitudes al API de CMS requieren un encabezado de autorización que contenga sus tokens de acceso. Para obtener detalles sobre cómo obtener credenciales de cliente y usarlas para recuperar tokens de acceso, consulte la Descripción general de Brightcove OAuth.

CONSEGUIR un video

Tenga en cuenta que no más de 10 ID de video se puede incluir.

Punto final: /videos/{video_id}[,{video2_id},{video3_id}]
