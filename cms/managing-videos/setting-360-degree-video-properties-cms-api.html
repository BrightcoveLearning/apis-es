Descripción general

Para identificar un video como un video de 360 °, se debe establecer la propiedad de proyección. Puede configurar esta propiedad usando el módulo de medios en Studio , o puede hacerlo mediante programación como se muestra en este tema.

Este ejemplo usa la aplicación Postman para actualizar las propiedades de video con la CMS API. Aprenda a comenzar con el Utilice Postman para solicitudes HTTP documento.

En este tema, realizará las siguientes tareas:

  • Obtenga las credenciales del cliente
  • Recuperar información de video
  • Establecer propiedades de video de 360 °

Obtenga las credenciales del cliente

Para trabajar con las API de Brightcove, necesitará las credenciales de cliente para la cuenta y las API que desea utilizar.

  1. Obtenga sus credenciales de cliente en Studio siguiendo las instrucciones en Administrar las credenciales de autenticación de API. En los pasos a continuación, realizaremos solicitudes de la API de administración de reproductores mediante Postman, por lo que sus credenciales deben tener al menos los siguientes permisos:

    • CMS: Lectura / escritura de video

    Puede agregar tantos permisos adicionales como desee para obtener credenciales que se podrán utilizar para una gama más amplia de solicitudes de API. También tenga en cuenta que obtiene credenciales que funcionarán para varias cuentas si lo desea.

    Para generar credenciales de cliente, debe tener permisos de administrador para Studio u obtener a alguien en su organización que tenga permisos de administrador para generar las credenciales por usted.

Recuperar información de video

Una vez que haya subido su video a Video Cloud Studio, estará listo para recuperar la información de video de la CMS API. Para obtener información sobre cómo cargar videos, consulte la Carga de videos usando el módulo de carga documento.

  1. Abra la aplicación Postman.
  2. Obtenga un nuevo token de acceso para su cuenta de Video Cloud y seleccione el Usar Token botón.
  3. Hacer una OBTENER solicitud a la CMS API, utilizando el Obtener video por ID sintaxis. Para hacerlo, ingrese la siguiente URL en el Ingrese la URL de la solicitud campo, sustituyendo su ID de cuenta por {account_id} y tu identificación de video para {video_id}: 
        https://cms.api.brightcove.com/v1/accounts/{account_id}/videos/{video_id}
  4. Deje el método como GET y seleccione Enviar:
  5. En el cuerpo de respuesta, debería ver el proyección propiedad.

Establecer propiedades de video de 360 °

Para indicar que un video es un video de 360 °, el proyección la propiedad debe establecerse en equirrectangular.

  1. Regrese a la aplicación Postman. Es posible que deba obtener un nuevo token de acceso.
  2. Seleccione PARCHE como el método HTTP, utilizando la misma sintaxis de URL que en la sección anterior.
  3. Haga clic en el Encabezados y proporcione el valor clave par Tipo de contenido: aplicación / json. Simplemente comience a escribir en las áreas de entrada y se le proporcionarán opciones entre las que puede elegir los valores indicados.
  4. Haga clic en el Cuerpo sección justo debajo de la URL, luego la crudo boton de radio:
  5. Para los datos sin procesar, ingrese el siguiente código JSON para el cuerpo (la captura de pantalla que sigue al JSON muestra cómo debe aparecer la solicitud): 
        {
        "projection" : "equirectangular"
    }
  6. Hacer clic Enviar
  7. En el cuerpo de la respuesta, ahora debería ver que el valor de la proyección Se ha establecido la propiedad.

    Ahora está listo para reproducir su video como un video de 360 °.
