Introducción

Ya sea que las pistas de texto sean ingeridas (estáticas o dinámicas) o remotas, puede usar un PARCHE solicitud para actualizar los metadatos asociados con las pistas de texto.

Notas

  • Los métodos explicados en este documento son para actualizar los archivos WebVTT agregados a sus videos. Puede ingerir archivos WebVTT en Video Cloud utilizando el API de ingesta dinámica o agregar archivos WebVTT remotos a tus videos usando la CMS API.

Autorización

Las solicitudes a la CMS API requieren un encabezado de autorización que contiene sus tokens de acceso. Para obtener detalles sobre cómo obtener credenciales de cliente y usarlas para recuperar tokens de acceso, consulte la Descripción general de Brightcove OAuth.

Las operaciones que necesitará para actualizar los metadatos de video son video-cloud/video/update o video-cloud/video/all.

Sintaxis

los toda la matriz debe ser parcheada , no solo la pista o el campo que está cambiando.

Recomendamos realizar la siguiente solicitud primero para obtener la matriz completa:

GET
https://cms.api.brightcove.com/v1/accounts/{account_id}/videos/{video_ID}

Actualización de pistas de texto

Al actualizar pistas de texto, algunos campos en Campos de pistas de texto puede tratarse de manera diferente según su tipo.

Campos de seguimiento de texto
Campo Tipo Pistas de texto remotas Pistas de texto ingerido estático Pistas dinámicas de texto ingerido
identificación Cuerda Valor de SÓLO LECTURA. Requerido para identificar el TT que necesita actualizarse.
ID de la cuenta Cuerda Se ignorarán las actualizaciones en este campo.
src Cuerda Obligatorio y limitado a 255 caracteres. Se ignorarán las actualizaciones en este campo.
srclang Cuerda Limitado a 32 caracteres.
etiqueta Cuerda Limitado a 255 caracteres.
amable Enum Los valores válidos son subtítulos , subtítulos , descripciones , capítulos , metadatos. Distingue mayúsculas y minúsculas.
tipo de Mimica Cuerda Limitado a 32 caracteres. Se ignorarán las actualizaciones en este campo.
asset_id Cuerda Valor de SÓLO LECTURA. Campo no aplicable para este tipo. Se ignorarán las actualizaciones en este campo.
in_band_metadata_track_dispatch_type Cuerda Campo no aplicable para este tipo. Se ignorarán las actualizaciones en este campo.
fuentes Formación Valor de SÓLO LECTURA.
defecto Booleano cierto o falso. No puede agregar varios predeterminado = verdadero a pistas de texto con diferentes srclang. Se ignorarán las actualizaciones en este campo.

Pedido

Aplicable a pistas de texto dinámicas, estáticas y remotas.

PATCH
https://cms.api.brightcove.com/v1/accounts/{account_id}/videos/{video_ID}

Solicitar ejemplos de cuerpo

Pistas de texto remotas

El cuerpo de la solicitud será un objeto JSON que contiene una matriz de pistas de texto:

{
	"text_tracks" : 
	[ 
		{ 
			"id" : "1234-1234-1234"
			"src" :     "http://learning-services-media.brightcove.com/captions/bc_smart_ja.   vtt",
			"srclang" : "ja", 
			"label" : "日本人", 
			"kind" : "captions", 
			"mime_type" : null, 
			"default" : true 
		}, 
		{ 
			"id" : "5678-5678-5678"
			"src" : "http://learning-services-media.brightcove.com/captions/bc_smart_es.   vtt",  
			"srclang" : "es", 
			"label" : "español", 
			"kind" : "descriptions",
			"mime_type" : null,
			"default" : false 
		} 
	] 
}
