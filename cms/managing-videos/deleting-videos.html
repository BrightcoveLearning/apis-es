Introducción

Probablemente necesite eliminar videos de vez en cuando, porque ya no son relevantes o por otras razones. Esto se puede hacer en Estudio o, como se explica a continuación, utilizando la API de CMS.

El proceso es sencillo, pero hay algunas cosas que debe tener en cuenta:

  • La eliminación de video no se puede deshacer.
  • Debido al almacenamiento en caché, los videos eliminados no desaparecen inmediatamente del sistema Video Cloud o de Internet. La única forma de hacer que un video no esté disponible de inmediato es eliminarlo de todos los reproductores.
  • Aunque puede eliminar varios videos, el número que se puede eliminar a la vez está limitado por la longitud máxima de URL para solicitudes de API. Generalmente, puede eliminar hasta 100 con una sola solicitud.

Detalles de la solicitud de API

Punto final

/v1/accounts/{account_id}/videos/{video1_id},{video2_id}...

Método HTTP

DELETE

Autenticación

Un token de acceso lo mismo que todos API de CMS solicitud.

Cuerpo de la solicitud

Ninguno.

Respuesta

204 No Content
