Introducción

Las propiedades de filtrado geográfico se pueden agregar al crear inicialmente el vídeo mediante una solicitud POST o mediante una solicitud PATCH para actualizar los metadatos de vídeo. Ver Actualizar videos usando API de CMS para ver un ejemplo de una solicitud de actualización.

En lugar de aplicar filtrado geográfico a nivel de video, o además de hacerlo, puede aplicar el filtrado geográfico a los reproductores de Brightcove a través de claves de política. Video Cloud examina las políticas de filtrado geográfico de vídeo y reproductor, y aplicará ambas. Si un país está excluido por cualquiera de las políticas, el vídeo no estará disponible en el país.

Tenga en cuenta que deben introducirse los códigos de país minúscula.

Algunos editores han solicitado la posibilidad de filtrar videos geográficos a bases militares estadounidenses, así como a Estados Unidos. Para apoyar esto, hay un Militar de Estados Unidos opción. Acceso al geofiltrado

El filtrado geográfico es un complemento de pago de Video Cloud. Si está interesado en comprar el paquete de filtrado geográfico, comuníquese con su administrador de cuentas.

Autorización

Solicitudes al API de CMS requieren un encabezado de autorización que contenga sus tokens de acceso. Para obtener detalles sobre cómo obtener credenciales de cliente y usarlas para recuperar tokens de acceso, consulte la Descripción general de Brightcove OAuth.

Las operaciones para las que necesitará permisos para actualizar los metadatos de video son video-cloud/video/update o video-cloud/video/all.

Sintaxis

geo es un objeto con tres propiedades:

Propiedad Tipo Necesario Comentarios
geo.countries Matriz de cadenas de códigos de país no Un Lista ISO 3166 de códigos de 2 letras en minúsculas para los países que se incluirán en la lista blanca o en la lista negra para ver el video
geo.exclude_countries Booleano no Si true , la visualización es no permitido en países incluidos en el geo.countries lista; Si false , la visualización es solo permitido en esos países.
geo.restricted Booleano no Si true el filtrado geográfico está habilitado para video

Pedido

    https://cms.api.brightcove.com/v1/accounts/{account_id}/videos/{video_ID}

Cuerpo de la solicitud:

El cuerpo de la solicitud será un objeto JSON que contendrá una matriz de abreviaturas de país que se incluirán o excluirán:

    {"geo" : {
        "restricted" : true,
        "exclude_countries" : false,
        "countries" : ["ph","in","my","ja"]
      }
    }
