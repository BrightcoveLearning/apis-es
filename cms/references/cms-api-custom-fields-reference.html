Campos Personalizados

Campo Tipo Descripción
max_custom_fields Número Número de campos personalizados que un editor puede crear en la cuenta.
custom_fields Formación Lista de objetos de campo personalizados en la cuenta
standard_fields Formación Lista de campos que son obligatorios o tienen una descripción. Los campos estándar que no tengan ninguno de los dos no aparecerán en esta matriz.

Mapa de campo personalizado

Para cada campo personalizado:

Campo Tipo Descripción
id Cuerda Nombre interno del campo
description Cuerda  
display_name Cuerda  
type Cuerda uno de 'enum' o 'cadena' (que se traduce como 'Lista' o 'Texto' en la interfaz de usuario)
enum_values Formación Matriz de valores de cadena válidos para el campo personalizado; solo presente cuando type = 'enum'
required Booleano Si este campo debe tener un valor antes de que se pueda activar el video
