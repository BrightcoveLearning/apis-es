Descripción general

los Analytics API le permite recuperar varios informes analíticos para sus videos de Video Cloud. Este inicio rápido le permitirá probar varias solicitudes diferentes para ver cómo se ven la URL de la solicitud y la respuesta.

  • Para obtener una explicación completa de la API, consulte la Descripción general de la API de Analytics y el Referencia de la API de Analytics
  • Si desea probar las solicitudes en su (s) propia (s) cuenta (s), pruebe nuestro Herramientas de prueba de API

Autorización

Las llamadas a la API están autorizadas mediante tokens de acceso OAuth2. El token se pasa con solicitudes en un encabezado de autorización:

    Authorization: Bearer {access_token}

Los tokens de acceso son de corta duración (5 min), por lo que debe obtener uno cada vez que realiza una solicitud

En el inicio rápido, todas las solicitudes serán para una cuenta de muestra de Brightcove, utilizando un apoderado que tiene las credenciales de cliente necesarias y obtendrá un token de acceso y luego realizará la solicitud.
