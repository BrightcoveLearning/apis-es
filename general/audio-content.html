Ingestión

Para agregar un archivo de audio a su cuenta de Video Cloud, use el API de ingesta dinámica o el Subir módulo. Su archivo de origen puede estar en formato MP3, M4A o WAV. El proceso de ingesta requiere dos solicitudes:

  1. Una solicitud de creación al API de CMS para crear el artículo en su catálogo de Video Cloud
  2. Una solicitud de ingesta a la API de ingestión para ingerir y transcodificar el archivo

Los detalles de estas solicitudes se encuentran en el Referencia de la API de ingesta dinámica.

Perfil de ingesta

Entrega dinámica

Si su cuenta está habilitada para Dynamic Delivery, no necesita un perfil personalizado para contenido de solo audio.

Ingesta heredada

Si su cuenta usa el sistema de ingesta heredado, use un perfil que cree representaciones de solo audio. Puedes usar el audio-only perfil que se incluye en el Perfiles estándar , o puede crear un perfil personalizado en el Configuración de la cuenta de Studio o usando el API de perfiles de ingesta.

Entregas

Las representaciones creadas en Video Cloud para su elemento de audio estarán en formato MP4 y HLS, solo con una pista de audio. El elemento se puede reproducir en un Brightcove Player como un video, como se explica debajo.

Nota: Para maximizar la cobertura de reproducción en múltiples plataformas / navegadores, recomendamos que los archivos de audio se transcodifiquen en versiones MP4 y HLS solo con pistas de audio: no puede especificar MP3 como formato de salida.

Póster y miniatura

Opcionalmente, puede agregar un póster y / o una imagen en miniatura a su archivo de audio. El póster se mostrará en el reproductor Brightcove durante la reproducción y la miniatura se mostrará en cualquier lista de reproducción a la que agregue el elemento.

Para agregar imágenes a su elemento de audio, puede editar la configuración del elemento en Studio o usa el API de ingesta dinámica.

Reproductor de música

Sus elementos de audio se reproducirán en cualquier Jugador de Brightcove. Durante la reproducción, la pantalla de video estará en blanco o mostrará la imagen del póster si añadido uno.

Si prefiere usar un reproductor de audio más minimalista sin la pantalla de video, puede crear un reproductor para ese propósito. Después de ti crea el jugador , configurar el tamaño en la siguiente manera:

  • Usa un tamaño fijo
  • Utilice una relación de aspecto personalizada
  • Establezca la altura en 80px (el ancho puede ser lo que quiera)
