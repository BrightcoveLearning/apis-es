Introducción

La API de reproducción tiene cuatro operaciones:

  • Obtener un video por id
  • Obtener un video por identificación de referencia
  • Obtener una lista de reproducción por id
  • Obtener una lista de reproducción por identificación de referencia
La API de reproducción está pensada para que la utilicen las páginas web o las aplicaciones móviles del lado del cliente; para la gestión general de medios o la integración de Video Cloud con sus sistemas, utilice la API de CMS en lugar de.

Si solo desea información sobre el video que se encuentra actualmente en el reproductor, puede usar el mediainfo objeto en lugar de realizar una llamada a la API de reproducción.

La mediainfo El objeto contiene información sobre el video cargado actualmente en el reproductor, mientras que la API de reproducción puede recuperar información sobre cualquier video. Para obtener más detalles, consulte la Metadatos de video de mediainfo documento.

Autenticación

Las solicitudes de la API de reproducción se autentican mediante una cadena de clave de política cifrada. El CodePen a continuación tiene una clave de política para una cuenta de muestra de Brightcove incluida, por lo que puede usarla si lo desea. Puede obtener claves de póliza para su cuenta utilizando el API de política o de uno de tus reproductores de Brightcove.

Instrucciones

Utilice CodePen a continuación para explorar la API:

  1. Comience simplemente haciendo solicitudes de videos y listas de reproducción para una cuenta de muestra de Brightcove para ver los resultados.
  2. A continuación, intente obtener su propia clave de política utilizando el API de política o de uno de tus reproductores de Brightcove.
  3. En el formulario CodePen a continuación, reemplace la identificación de la cuenta, la clave de la política, la identificación del video y la identificación de la lista de reproducción con sus propios valores.
  4. Intente modificar el CSS para personalizar la visualización de los datos del video.
  5. Intente modificar JavaScript para obtener videos y listas de reproducción por identificación de referencia en lugar de identificación.
