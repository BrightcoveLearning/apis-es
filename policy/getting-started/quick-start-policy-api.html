Introducción

Notas:
  • La clave de política solo se usa para controlar videos que se almacenan en Video Cloud. Si está utilizando el reproductor Brightcove independiente, no es necesario acceder ni recuperar la clave de política.
  • Las claves de política generadas para los nuevos reproductores de Brightcove no están habilitadas para búsqueda, restricción de dominio, filtrado geográfico ni requieren SSAI de forma predeterminada. Debe configurar esto manualmente mediante la API de políticas (o esta aplicación de muestra). Para obtener más detalles, consulte la Descripción general de la API de políticas documento.
Esta muestra y el código asociado se proporcionan como una guía para su desarrollo de producción. El código no está certificado para uso en producción ni se admite oficialmente.

Obtener credenciales

Para utilizar la API de políticas, necesitará las credenciales adecuadas. En este inicio rápido, trabajará en una cuenta de prueba de Brightcove, por lo que no necesitará completar este paso, pero tenga en cuenta que deberá obtener las credenciales de cliente cuando trabaje en su propia cuenta.

  1. Para obtener credenciales, siga las instrucciones en Administrar las credenciales de la API. Esta es la forma recomendada de obtener sus credenciales de cliente. Los permisos que necesitará son Lectura del reproductor y Lectura / escritura del reproductor:

    También puede obtener credenciales mediante la API de OAuth, siguiendo las instrucciones de una de las guías que se enumeran a continuación:

    • OAuth: Obtener credenciales de cliente mediante CURL
    • OAuth: Obtenga las credenciales del cliente con Postman

Ver la pluma Crear aplicación de clave de política por Brightcove Learning Services ( rcrooks1969 ) en CodePen.

Usando el CodePen

A continuación, se incluyen algunos consejos para utilizar eficazmente el CodePen anterior:

  • Cambie la visualización real del reproductor haciendo clic en el Resultado botón.
  • Haga clic en el HTML / CSS / JS botones para mostrar UNO de los tipos de código.
  • Hacer clic Editar en CodePen en la esquina superior derecha para bifurcar este CodePen en su propia cuenta.
  • Encuentre todo el código asociado con esta muestra en este Repositorio de GitHub.

Código de proxy

Para crear su propia versión de la aplicación de muestra en esta página, debe crear y alojar su propio proxy. (Los proxies utilizados por Brightcove Learning Services solo aceptan solicitudes de dominios de Brightcove). Un proxy de muestra, muy similar al que usamos, pero sin las comprobaciones que bloquean las solicitudes de dominios que no son de Brightcove, se puede encontrar en este repositorio de GitHub. También encontrará instrucciones básicas para usarlo allí, y una guía más elaborada para crear aplicaciones alrededor del proxy en Usando las API REST.
